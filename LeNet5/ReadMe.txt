#ReadMeFile
Dataset is downloaded fro char74 dataset.
Dataset Consist of images of Letters A-Z and numbers 0-9 each of which has approx 1000 images in the dataset.
Dataset is splitted int train and test containing containing 36 different Categories.
Model is a CNN classifier made by using keras library.
Currently the no.of.epochs is 3 for more accuracy no of epochs can be increased or by increasing the hidden layers.
Parameter tuning can also be done if someone has a fast processor.
